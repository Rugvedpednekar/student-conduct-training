Overdue Sanctions Tool

What overdue sanctions are and why they matter

Overdue sanctions are assigned outcomes that have passed their listed completion deadline and still need follow-up. Tracking them carefully supports outreach, hold decisions, and consistent case closure.

Reminder

Review each row carefully before outreach. Some sanctions depend on outside completion confirmation, some may already be in progress, and some may require supervisor context.

Step-by-step: pull the report from Maxient

  1. Sign in to Maxient and open the reporting or export area used for sanctions tracking.
  2. Run the Overdue Sanctions report or the equivalent overdue sanctions export view.
  3. Confirm that fields such as File ID, names, SID, email, sanctions, deadline fields, and hold status are present.
  4. Export the report as .xlsx and save it locally.
  5. Upload the file below, choose the date range, review the preview, and download the cleaned workbook.

What to review first

  • Student identifier quality: Full Name, Student ID, and Email.
  • Assigned sanctions text so follow-up remains clear and readable.
  • Next Deadline and Next Deadline Reason for outreach timing.
  • Hold in Place and File ID for internal tracking and coordination.

What this tool does

  • Filters the uploaded report using your selected From and To dates.
  • Builds a cleaner output with key fields such as name, ID, email, and assigned sanctions.
  • Shows a preview before download so you can confirm the results.
  • Creates a separate workbook and does not overwrite the original file.
Important

Upload only a valid .xlsx Maxient export. The original upload remains unchanged.

Quick Checklist
  • Report exported from Maxient today or for the intended review cycle
  • File saved as .xlsx
  • Date range selected correctly before processing
  • Preview reviewed before download
Suggested output check

Before download, confirm the preview includes the correct students, the right date range, and readable sanctions text.
