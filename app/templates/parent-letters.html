Parent Letters

This section explains how to find the correct parent-letter case, generate the letter, print the envelope and the letter, complete the mailing process, and update the sanction correctly.

Module Focus Parent Letter Workflow

Parent letters must be handled carefully, with accurate records, correct printing steps, and proper mailing follow-up.

Parent Letter Workflow Video

Video walkthrough: Finding the parent-letter case, generating the letter, printing the envelope, and printing the final letter.

Important Practice

Parent letters should be factual, policy-based, and carefully reviewed. Do not add unnecessary case detail, assumptions, or personal wording that goes beyond the approved office purpose.

Step-by-Step Parent Letter Process

1. Find the Parent Letter Case

  1. Go to Analytics.
  2. Select Code 53, which is Incomplete Sanctions by Cases.
  3. For the time range, select the current academic year.
  4. Use reported date.
  5. Make sure the case type is Student Conduct.
  6. Run the report.
  7. Sort the results by next deadline to find recent cases.
  8. Review the Sanction and Actions section to identify where letter to parent is written.

2. Open the Case and Confirm the Letter Type

  • Click the File ID to open the case in a new tab.
  • Check the sanction details carefully.
  • Confirm what type of parent letter is required. For example, it may be a drug violation parent letter.

3. Generate the Parent Letter

  1. Go to Assignments.
  2. Click Save Updates so the letter section becomes available.
  3. Open Letters.
  4. Select the correct parent letter template. In the example, the drug parent letter is code 7.
  5. The signer should always be Dave.
  6. Build the letter and review the preview carefully.
  7. Confirm all details are correct before printing anything.

4. Print the Envelope

  1. Choose to generate an envelope.
  2. Keep the optional field blank if it is not needed.
  3. Always select the permanent address.
  4. Generate the envelope.
  5. When printing, open More Settings.
  6. Set the paper size to Envelope #10.
  7. Check the preview and then print the envelope.

Printer help: How to customize printer settings for Envelope #10

5. Print the Parent Letter

  • Create a printable copy of the letter.
  • Refresh if needed to get the printable copy.
  • Print the letter using normal letter settings.
  • Adjust page selection if needed before printing.

6. Prepare the Letter for Mailing

  1. After you are done printing the envelope, give it to Dave.
  2. Dave will sign it and give it back to you.
  3. Place the parent letter inside the envelope and close it.
  4. Write the code 7121 on the top right corner of the envelope.
Do Not Forget This

The code 7121 must be written on the top right corner of the envelope. If you do not write this code, the mailing office at the University will not accept it.

7. Mail the Parent Letter

  • Take the completed envelope to the 1st floor of GSU.
  • Give it to the mailing office.

8. Update the Sanction After Mailing

After the letter has been mailed, go back to the case and update the Parent Letter sanction as Completed.

This was not shown in the video, but it is an important final step and should not be forgotten.

Communication Cautions

  • Keep language factual and policy-based.
  • Avoid over-disclosure of case details.
  • Verify the correct student and correct address before preparing anything.
  • Use the correct letter type for the case.
  • Do not skip the mailing-office code.
  • Do not forget to mark the sanction completed after mailing.
