Office Templates

This page is the template reference area for Student Conduct communication. Use it to understand which template should be used, when it should be used, and what kind of message it supports.

Module Focus Communication Tools

Use the correct template for the correct workflow so communication stays consistent, professional, and accurate.

How to Use This Page

Each template below should help you understand both the purpose of the message and the situation where it should be used. Before sending any template, confirm that the facts, recipients, timing, and case details are correct.

Important Reminder

Templates are starting points. They help maintain consistency, but they should still be reviewed carefully before use so that the message matches the case and the office’s communication standards.

Template Categories

Hearing Notice Template

Use for: Sending a student their hearing date, time, and instructions.

Where to use it: When a case has been assigned and the student must be formally notified of the hearing.

Hearing Reminder Template

Use for: Reminding a student shortly before a scheduled hearing.

Where to use it: When you need a clear, professional reminder about an upcoming meeting.

No-Show Follow-Up Template

Use for: Following up after a student does not attend a scheduled hearing.

Where to use it: When a student misses a conduct meeting and a follow-up communication is required.

Sanction Completion Reminder

Use for: Reminding a student about an incomplete or overdue sanction.

Where to use it: During sanctions follow-up when completion has not yet been verified.

Parent Letter Draft Placeholder

Use for: Drafting a parent notification letter after policy criteria and approval are confirmed.

Where to use it: In approved parent-notification workflow only.

Internal Follow-Up Template

Use for: Sending an internal message to a supervisor or staff member about a workflow item.

Where to use it: When you need to confirm a status, flag an issue, or request direction.
