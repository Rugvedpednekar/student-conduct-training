Case Handling

This section walks through the case lifecycle from intake through closure. A Graduate Assistant should understand how to create and review cases, support hearings, keep documentation accurate, and follow through on post-hearing responsibilities without skipping workflow steps.

Workflow Focus Case Lifecycle

Each step should be completed carefully before the case moves forward.

Case Lifecycle

Use this structure to keep case processing accurate, organized, and consistent.

Important Reminder

There are two main report paths you may work with: Maxient Pending IRs / Incoming Reports and Public Safety Reports. If the same incident has multiple reports, they should be merged into one case instead of creating duplicates.

Part 1: How to Create a Case from Pending IRs

Video walkthrough: Creating a Student Conduct case using Maxient Pending IRs.

  1. Go to Pending IRs and sort the data to find the recent reports.
  2. Sometimes there may be more than one report for the same incident.
  3. If the same incident appears multiple times, create only one case and merge the related reports later.
  4. Select a report, click Action, and choose Create Case and Go.
  5. The case type should always be Student Conduct.
  6. Select the appropriate area such as Public Safety or Residential Life based on the report source.
  7. Confirm that the student shown is the correct student involved in the incident.
  8. The Home Office should always be Student Conduct.
  9. Assign the hearing officer based on Dave’s direction.
  10. Add the correct charge or issue and add it to the involved student.
  11. If alcohol is clearly reportable, use the appropriate alcohol-related violation.
  12. Continue through the required screens and click Save and Create the Case.

Part 2: How to Create a Case from a Public Safety Report

Video walkthrough: Creating a Student Conduct case from a Public Safety report.

  1. After receiving the Public Safety report, click Create Case and open a new case.
  2. Enter the student ID from the report.
  3. Search for the student and confirm the student is correct before continuing.
  4. Select the student and click Get Started.
  5. The case type should always be Student Conduct.
  6. The case is referred by Public Safety.
  7. Check the report carefully and make sure the date and time match exactly.
  8. Enter the location and the specific location, such as the room number.
  9. Select the student as an involved student.
  10. The Home Office should be Student Conduct.
  11. Assign the case based on Dave’s direction.
  12. Add the correct charges and add them to the involved student.
  13. If the report clearly supports a reportable violation, choose the correct disciplinary or law/policy violation.
  14. Upload the report or required case material as part of the workflow.
  15. When finished, click Save and Create Case.
Manual Entry Reminder

Creating a case from a Public Safety report involves more manual work than creating one from Pending IRs. Be especially careful with date, time, location, student ID, and charges.

Part 3: What to Do When There Are Multiple Reports for the Same Incident

Video walkthrough: Merging multiple reports that belong to the same incident.

  1. If there are two or more reports for the same incident, do not create multiple separate cases.
  2. Find the case that has already been created and copy the case ID number.
  3. Click Actions.
  4. Select Merge with IR for the Case.
  5. Enter the case number.
  6. Select the matching report.
  7. Click Go.
  8. Repeat this process for any other report tied to that same incident.
Merge Rule

One incident should generally have one case. Additional reports for the same incident should be merged into the existing case.

Part 4: Schedule the Hearing and Send the Hearing Notification

Video walkthrough: Scheduling a hearing and sending the hearing notification email.

  1. Go to Home and open Unheard Cases or Unresolved Cases.
  2. Click the File ID to open the student case.
  3. Review the student record and the EFC.
  4. Go to Assignments.
  5. Check the student schedule before selecting a hearing date and time.
  6. Schedule the hearing around the student’s classes whenever possible.
  7. Students must receive 48 business hours notice before the hearing.
  8. Enter the hearing date, time, and location. The location is typically GSU Room 307.
  9. Click Save Updates.
  10. Go to Letters and choose the hearing notification template.
  11. Choose yourself as the signer and review the draft carefully.
  12. Send the hearing notification through Maxient.

Part 5: After the Hearing

Video walkthrough: Completing the CRF, entering findings, applying sanctions, and finalizing post-hearing workflow.

Complete the CRF

  1. There are two ways to open Complete CRF:
    • From Unheard Cases → Action → Complete CRF
    • From File ID → Assignments → Save Updates → Complete CRF
  2. The Resolution Type is normally Administrative Student Conduct Hearing.
  3. The Resolution Date should be the date the resolution is given.
  4. In administration, use your name as the hearing officer.

Enter Findings

  • Select whether the student was found responsible or not responsible.
  • Findings should match the hearing outcome and the charges.

Apply Sanctions

  • Select sanctions based on the violation.
  • Use the minimum sanction guideline to choose the correct sanctions.
  • Set the due date, which is normally about two weeks after the resolution date.
  • If probation is assigned, select the correct probation end date.
  • If the student is under 21 and the sanction requires it, include the parent-letter sanction.

Add the Rationale and Final Deadline

  • Enter the rationale for the resolution.
  • Review the CRF carefully before finalizing.
  • The next deadline should be the final deadline for sanction completion.

Use Rapid Resolve When Appropriate

  • If the same resolution is being applied to multiple students in the same case, use Rapid Resolve.
  • Select the students who should receive the same resolution.
  • Then finalize the CRF.
Final Review Reminder

Check the sanction guidelines carefully. Make sure sanctions, deadlines, rationale, and findings are all correct before using Final CRF.

Case Reminders

Important

Do not guess on case direction. If anything is unclear, ask Dave before moving forward.

Main Case Paths

  • Pending IRs / Incoming Reports
  • Public Safety Reports
  • Merge duplicate incident reports

After the Hearing

  • Complete the CRF
  • Enter findings
  • Apply sanctions correctly
  • Review rationale and deadlines
  • Use Rapid Resolve when appropriate

Documentation Rule

If it is not documented clearly in the case workflow, it should not be treated as complete.
